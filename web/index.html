About

This program, Voxelmath, is intended for marking images for texture analysis. It analizes exactly what you see on the screen (greyscale values from 0 to 255), and not Houndsfeld Units or other variables.
Put your images in a subdirectory of 

web/mriimages/
directory of this program, after choosing the type of image labelling, then refresh the page.
